# ClipMate PRD · 产品需求文档

**版本：** v1.0  
**日期：** 2026-03  
**技术栈：** Electron + Vue3 + Vite · 仅 macOS

---

## 一、产品概述

### 1.1 产品定位

ClipMate 是一款面向 macOS 用户的专业剪贴板管理工具。它作为系统剪贴板的增强层，自动记录用户所有复制过的内容，并提供搜索、分类、预览、顺序粘贴等功能，帮助用户摆脱"复制覆盖"的困境，提升日常工作效率。

目标用户：开发者、设计师、写作者、内容创作者、频繁处理多任务的知识工作者。

### 1.2 核心价值

- **不丢失**：再也不会因为新的复制操作覆盖掉之前的内容
- **找得到**：通过全文搜索、类型过滤、OCR 等手段，快速定位任何历史内容
- **用得顺**：一键粘贴、顺序粘贴、快捷键操作，融入原有工作流，无感知
- **放得心**：数据本地存储，隐私可控，敏感应用可排除

### 1.3 竞品参考

Paste（pasteapp.io）— 本产品的主要参考对象，功能子集实现。

```

├── 数据库读写
├── iCloud 同步
├── 系统快捷键注册
├── 窗口管理

├── 历史记录列表
```

### 2.3 数据模型

**ClipItem（剪贴板条目）**

| 字段            | 类型        | 说明                                          |
| --------------- | ----------- | --------------------------------------------- |
| id              | TEXT (UUID) | 主键                                          |
| type            | TEXT        | text / richtext / link / image / file / color |
| content         | TEXT / BLOB | 主要内容（文本 or base64）                    |
| plain_text      | TEXT        | 纯文本版本（用于搜索）                        |
| ocr_text        | TEXT        | 图片 OCR 结果（可为空）                       |
| source_app      | TEXT        | 来源 App Bundle ID                            |
| source_app_name | TEXT        | 来源 App 名称                                 |
| title           | TEXT        | 用户自定义标题（可为空）                      |
| thumbnail       | BLOB        | 缩略图（图片 / 链接预览用）                   |
| link_meta       | TEXT (JSON) | 链接元数据 { title, description, favicon }    |
| is_confidential | INTEGER     | 是否为机密内容（0/1）                         |
| created_at      | INTEGER     | 创建时间戳（ms）                              |
| updated_at      | INTEGER     | 更新时间戳（ms）                              |

## 三、功能需求

### 3.1 核心剪贴板引擎

**FR-01 自动捕获剪贴板内容**

系统在后台持续监听系统剪贴板变化。当检测到新内容时，自动识别类型并保存到本地数据库。支持的内容类型：纯文本、富文本（RTF / HTML）、链接 URL、图片（PNG / JPG / GIF / WebP）、文件路径引用、颜色值（16进制）。

去重规则：若新复制内容与数据库中最新一条完全相同，则不重复写入，仅更新时间戳。

**FR-02 暂停 / 恢复收集（Pause Paste）**

用户可通过菜单栏图标或快捷键随时暂停 / 恢复剪贴板监听。暂停期间的复制内容不会被记录。状态在菜单栏图标上有视觉区分。

**FR-03 Direct Paste（直接粘贴）**

用户在 ClipMate 面板中单击历史条目时先选中该条目；双击条目或按回车时，内容立即粘贴到面板呼出前的最后活跃应用窗口，面板随即关闭。无需手动 Cmd+V。

**FR-04 纯文本粘贴**

右键菜单或快捷键可选择"粘贴为纯文本"，去除原始富文本 / HTML 格式后粘贴。

**FR-05 应用排除（黑名单）**

用户可在设置中添加应用到黑名单（通过 Bundle ID 或从已安装应用中选择）。黑名单应用中的复制操作不会被记录。典型用途：密码管理器（1Password、Bitwarden）。

---

### 3.2 历史记录

**FR-06 历史记录列表**

主面板展示所有历史记录，按时间倒序排列。每条记录展示：内容类型图标、内容预览（截断）、来源 App 图标、相对时间（"3分钟前"）。

**FR-07 历史记录上限与自动清理**

用户可在设置中配置历史记录最大保留数量（100 / 500 / 1000 / 无限制）或按时间（7天 / 30天 / 90天 / 永久）。超出限制时自动删除最旧的条目。

**FR-08 手动删除**

右键菜单支持删除单条记录；多选后支持批量删除；设置中支持一键清空所有历史。

---

### 3.3 搜索

**FR-09 实时全文搜索**

在搜索框输入关键词，即时搜索历史记录的纯文本内容、item 标题、链接 URL、OCR 识别文本。结果实时更新，关键词高亮显示。

**FR-10 类型过滤**

搜索结果可按内容类型筛选：全部 / 文本 / 链接 / 图片 / 文件 / 颜色。

**FR-11 来源应用过滤**

可按来源应用过滤，展示某特定 App 中复制过的所有内容。

**FR-12 日期范围过滤**

可按日期过滤：今天 / 本周 / 自定义日期范围。

**FR-13 图片 OCR 搜索**

对保存的截图 / 图片内容，后台异步进行 OCR 文字识别（调用 macOS Vision framework），识别结果存入数据库，纳入全文搜索索引。

---

### 3.4 预览与编辑

**FR-14 快速预览（Quick Look）**

选中条目后按空格键，弹出预览窗口。不同类型各有预览形式：

- 文本：完整文本展示，支持滚动
- 图片：全尺寸预览，显示分辨率 / 大小信息
- 链接：内置 WKWebView 展示网页
- 文件：系统 Quick Look 预览
- 颜色：大块色块 + HEX / RGB / HSL 值

**FR-15 文本编辑**

文本类条目从主面板的编辑入口进入独立预览窗口编辑态；预览窗口中文本类条目可直接编辑，Cmd+S 保存，修改立即同步（若开启同步）。

**FR-16 链接编辑**

链接类条目可在预览窗口中修改 URL，保存后更新记录并重新抓取元数据。

**FR-17 图片编辑**

图片预览中支持旋转（顺时针 / 逆时针 90°）。

**FR-18 颜色调整**

颜色值条目在预览中可打开内置颜色选择器调整颜色，保存后更新记录。

**FR-19 OCR 文字提取**

图片预览中提供"提取文字"按钮，将 OCR 结果复制到剪贴板或创建新文本条目。

**FR-20 条目重命名**

右键菜单 / 预览窗口中可为任意条目设置自定义标题，用于显示和搜索。

**FR-21 新建条目**

支持不通过复制，直接在 ClipMate 内手动创建文本或链接条目（Cmd+N）。

---

### 3.5 Paste Stack（顺序粘贴）

**FR-22 Stack 模式**

用户进入 Stack 模式后，每次复制操作都会将内容追加进队列，而非覆盖。激活 Stack 粘贴后，内容按队列顺序逐一粘贴到目标应用（每次 Cmd+V 消耗队列头部一条）。

**FR-23 Stack 队列可视化**

面板中展示当前 Stack 队列的所有内容和顺序，支持手动调整顺序或移除某一项。

**FR-24 退出 Stack 模式**

用户可随时通过快捷键或按钮退出 Stack 模式，清空队列，恢复正常剪贴板行为。

---

### 3.6 图片增强

**FR-25 图片拖拽到外部应用**

从 ClipMate 面板中可直接将图片条目拖拽到 Finder、Figma、浏览器等支持文件拖入的应用。

**FR-26 图片作为文件粘贴**

右键菜单提供"粘贴为文件"选项，将图片写入临时目录后以文件形式粘贴，支持上传到网页表单。

---

### 3.7 隐私与安全

**FR-28 屏幕共享时隐藏**

设置中可开启"屏幕共享时隐藏 ClipMate 窗口"，开启后在系统检测到屏幕录制 / 共享时，ClipMate 面板从共享画面中隐藏（Window Level 设置）。

**FR-29 忽略机密剪贴板内容**

若系统剪贴板数据被来源 App 标记为 concealed（如密码管理器自动标注），ClipMate 自动不记录该条内容。

**FR-30 本地优先，无强制云同步**

iCloud 同步为可选功能，默认关闭。所有数据优先存储在本地 SQLite 文件。

---

### 3.8 iCloud 同步

**FR-31 可选 iCloud 同步**

用户可在设置中手动开启 iCloud 同步。开启后，历史记录在同一 Apple ID 下的 Mac 设备间同步（本期不同步至 iOS）。

**FR-32 同步状态展示**

同步时在菜单栏图标或设置页展示同步状态（同步中 / 上次同步时间 / 同步失败提示）。

**FR-33 冲突处理**

同步冲突采用"最后写入胜出"（last-write-wins）策略，以 updated_at 时间戳为准。

---

### 3.9 快捷键

**FR-34 可自定义全局快捷键**

以下操作支持用户在设置中自定义全局快捷键：

| 操作                           | 默认快捷键        |
| ------------------------------ | ----------------- |
| 呼出 / 隐藏主面板              | Cmd + Shift + V   |
| QuickPaste（直接粘贴最近一条） | 自定义            |
| 纯文本粘贴                     | 自定义            |
| 进入 / 退出 Stack 模式         | 自定义            |
| 暂停 / 恢复收集                | 自定义            |
| 新建文本条目                   | Cmd + N（面板内） |
| 快速预览                       | 空格（面板内）    |
| 搜索聚焦                       | Cmd + F（面板内） |
| 关闭主面板                     | Esc（面板内）     |

---

### 3.10 界面与体验

**FR-36 菜单栏常驻图标**

ClipMate 在 macOS 菜单栏常驻显示图标。左键点击呼出主面板；右键点击展示快捷菜单（暂停收集 / 设置 / 退出）。

**FR-37 窗口行为**

主面板默认出现在当前鼠标所在的显示器上，位于屏幕中下方。失焦后自动隐藏。

**FR-39 深色 / 浅色模式**

自动跟随 macOS 系统外观设置，无需手动切换。

**FR-40 虚拟列表**

历史记录列表使用虚拟滚动渲染，确保在记录数量达到数万条时也能保持流畅滚动。

## 四、非功能需求

### 4.1 性能

- 主面板从快捷键触发到完全展示：< 150ms
- 剪贴板监听延迟（复制到记录入库）：< 300ms
- 搜索结果首次响应：< 100ms（本地 SQLite 全文检索）
- 图片 OCR 后台处理，不阻塞 UI
- 历史记录 10,000 条以内列表滚动帧率不低于 60fps

### 4.2 稳定性

- 主进程崩溃时自动重启
- 数据库写入使用事务，避免数据损坏
- 异常退出后下次启动自动恢复状态

### 4.3 兼容性

- 最低支持：macOS 13 Ventura
- 推荐：macOS 14 Sonoma 及以上
- 支持 Apple Silicon（arm64）和 Intel（x64）
- 通用二进制（Universal Binary）打包

### 4.4 隐私合规

- 无任何用户数据上传到第三方服务器
- iCloud 同步通过用户自己的 Apple ID，数据归用户所有
- 首次启动需请求辅助功能权限，展示明确的权限说明
- 符合 Apple App Store 隐私标签要求
